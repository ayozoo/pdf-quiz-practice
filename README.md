# AWS 刷题神器 (Exam Web) 🚀

这是一个专为 **AWS 认证考试**（如 SAA-C03）打造的刷题小网站，主打一个**轻量、好用、随地刷**。

> **💡 特别说明**：本项目后端的 PDF 解析逻辑是专门针对某套 AWS 题库格式编写的正则表达式。如果你上传其他格式的 PDF，大概率会解析失败。建议只用来刷 AWS 的题。

## ✨ 核心亮点

- **🎯 专为 AWS 题库优化**
  - 完美适配 AWS 题目结构：单选、多选、长题干、代码块。
  - 自动提取解析部分，刷题时随时查看答案详解。

- **📱 手机电脑无缝切换**
  - **电脑端**：经典左右布局，左边选题目，右边看解析，大屏刷题超爽。
  - **手机端**：自动变身 App 模式！底部导航、抽屉式菜单、大字体，躺在床上也能单手刷题。

- **💾 进度永不丢失**
  - 不需要登录，不需要注册。
  - **全自动保存**：选个选项、翻个页，进度立马存到浏览器里。
  - **智能识别**：哪怕你没点“提交”，只要选了答案，系统就认你做过这道题了，主打一个省心。

- **🔌 局域网无痛访问**
  - 只要手机和电脑连着同一个 WiFi，输入电脑 IP 就能直接访问。
  - 已完美解决 iOS 剪贴板限制、跨域等各种奇奇怪怪的问题。

## 🛠️ 怎么跑起来？

前提：电脑上得有 **Node.js** (建议 v18+)。

### 1. 下载与安装
```bash
# 进入项目目录
cd exam-web

# 一键安装所有依赖（前端+后端）
npm install
```

### 2. 启动服务
```bash
# 一句命令，前后端全起飞
npm start
```
看到控制台不报错，就是成功了！
- **前端页面**：打开浏览器访问 `http://localhost:5173`
- **后端接口**：运行在 `http://localhost:3000`

### 3. 手机怎么连？
1. 确保手机和电脑连的是同一个 WiFi。
2. 查一下你电脑的局域网 IP（Mac 输入 `ifconfig`，Windows 输入 `ipconfig`，找类似 `192.168.x.x` 的）。
3. 手机浏览器输入：`http://<你的电脑IP>:5173`。
4. **起飞！** 🛫

## ⚠️ 小贴士 (注意事项)

1.  **进度同步问题**
    - 进度是存在**当前浏览器**里的 (LocalStorage)。
    - 所以：**手机刷的进度电脑上看不用，电脑刷的手机上也看不到**。各自独立，互不干扰。

2.  **iOS 复制功能**
    - 如果你在 iPhone 上点“复制题目”没反应，不用慌。
    - 我们内置了“黑科技”兼容方案，会自动弹出一个隐形输入框帮你复制，放心用。

3.  **题库哪里来？**
    - 在“管理页面”上传 AWS 题库 PDF。
    - **注意**：后端解析器是写死的正则（专门匹配 `Question #`、`Correct Answer:` 这种格式）。不要乱传奇怪的 PDF，会报错的。
